## Install

```bash
go get github.com/gookit/goutil/ext
```

## Package: lcache 

`lcache` 简单、协程安全的本地内存缓存包，提供 TTL 过期支持、LRU 淘汰策略和文件持久化功能。

> [!NOTE]
> 更多信息请查看: [./lcache](./lcache/README.md)
